# AI 學習營 - 第二天 (Day-002)

[![Project Status: Active](https://img.shields.io/badge/Project%20Status-Active-brightgreen.svg)](https://github.com/asia17242/Ai-Study)
歡迎來到 AI 學習營的第二天。本階段目標是利用最新一代的前端技術棧，構建具備高性能數據視覺化能力的儀表板系統。

## 🚀 專案概覽

此專案展示了一個高性能、現代化的儀表板介面。它深度集成了 React 19 與 Next.js 16 (App Router)，並配合 Tailwind CSS v4 的原生效能，確保極速的渲染與流暢的交互體驗。

## 🛠 技術棧

- **框架:** [Next.js 16.2.x](https://nextjs.org/) (Experimental App Router)
- **核心庫:** [React 19.2.x](https://react.dev/)
- **樣式:** [Tailwind CSS v4](https://tailwindcss.com/)
- **數據視覺化:** [Recharts 3.8.x](https://recharts.org/)
- **圖標庫:** [Lucide React](https://lucide.dev/)

## 📁 目錄結構

- `frontend/`: 核心 Next.js 應用，包含 Dashboard 組件與數據分析邏輯。

## 📦 快速開始

1. **進入前端目錄:**
   ```bash
   cd frontend
   ```
2. **使用 npm 安裝依賴 (建議 Node 20+):**
   ```bash
   npm install
   ```
3. **啟動本地開發環境:**
   ```bash
   npm run dev
   ```
